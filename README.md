# CNN-Based-Alternative-Medicine-Recommender
The following repo contains, a CNN based medicine recommendation system that is mainly developed using Python as core programming language because of its simplicity and fast performance and is widely using in Machine Learning nowadays, 

Project Structure:

Frontend: HTML5, CSS3, Javascript

Backend: Python with Flask Framework

Database: MySQL

Python Libraries: Pandas, Numpy, Matplotlib, Scikit Learn

Deep Learning Model: CNN (Convolutional Neural Networks) - Sequential CNN

Dataset: Tabular csv with 5000 rows and 15 columns

DL Libraries: Tensor Flow, Keras API

EPOCHS Used: 50

Filters Used: 64

Dense Layer: 256 Neurons to prevent overfitting

Testing Probability Used: 0.8

Training Probability Used: 0.2

As we know that cnn's are commonly seen in image classification, audio and video extraction, A CNN is a deep learning model/algorithm that is going to train on labelled dataset which is also known as the supervised learning for obtaining classification by recognising complex patterns.
Here i have used the "ReLu Activation" Function in Forward Propagation and for Loss Function have chosen "Cross Entropy"
